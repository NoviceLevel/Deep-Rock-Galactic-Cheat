# Deep Rock Galactic Cheat

A game cheat for Deep Rock Galactic based on OmegaWare Framework.

## Features

### Player Modifications
- Godmode
- Run Speed
- Jetpack (Fly)
- Name Changer

### Weapon Modifications
- Infinite Ammo
- No Overheating
- No Reload
- No Recoil
- No Grapple Restrictions

### Aimbot
- Auto Fire
- Magic Bullet
- Multi-Target
- FOV Adjustment

### ESP
- Enemies / Friendlies / Players
- Objectives
- Special Structures
- Resource Chunks
- Accurate Box / Box
- Health Bar / Armor Bar
- Distance Display

## Hotkeys

- `Insert` - Toggle Menu
- `Home` - Toggle Console
- `End` / `Delete` - Unload

## Supported Languages

- English
- Deutsch (German)
- Polski (Polish)
- 中文 (Chinese)

- Visual Studio 2022
- Windows SDK
- C++20
